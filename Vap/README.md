# Папка VaP — портфолио

Положите сюда фото и видео с мероприятий.

**Поддерживаемые форматы:**
- Фото: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`
- Видео: `.mp4`, `.webm`, `.mov`

**Как обновить сайт:**

1. Скопируйте файлы в эту папку (можно .MOV с iPhone)
2. Запустите в корне проекта: `npm run media`
3. Скрипт автоматически конвертирует MOV → MP4 для Chrome/Edge
4. Обновите страницу в браузере (Ctrl+F5)

Имена файлов станут подписями (дефисы и подчёркивания заменятся на пробелы).
